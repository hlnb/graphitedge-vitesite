# Welcome to GraphitEdge Tutorials

## Why Learn with GraphitEdge?

At GraphitEdge, we believe learning web development should be accessible, practical, and enjoyable. Our tutorials are crafted by Helen B, a professional web developer and educator with over 20 years of experience in both teaching and building real-world applications.

### Our Teaching Philosophy

1. **Learn by Doing**

   - Every tutorial includes interactive examples
   - Practice exercises based on real projects
   - Hands-on approach to reinforce concepts

2. **Clear Progression**

   - Structured learning path
   - Build on concepts gradually
   - Each tutorial connects to the next

3. **Real-World Focus**
   - Examples from actual projects
   - Industry best practices
   - Current and relevant technologies

## Meet Your Instructor: Helen B

Hi! I'm Helen, and I'm passionate about helping others learn web development. My journey began in the early days of the web, and I've evolved with the technology ever since. What makes my approach different?

- **Practical Experience**: I actively work on web projects while teaching
- **Student-Focused**: My tutorials are based on real student feedback and questions
- **Clear Communication**: Complex concepts explained in simple terms
- **Ongoing Support**: Regular updates to keep content current

## Getting Started

1. **Set Up Your Environment**

   - Start with [Development Setup](/tutorials/tools/development-setup)
   - Install essential tools and extensions
   - Configure your workspace

2. **Choose Your Learning Path**

   - Complete beginner? Start with [HTML 101](/tutorials/basics/html-101)
   - Know the basics? Jump into [Vue Basics](/tutorials/vue/vue-basics)
   - Want to improve your CSS? Try [Tailwind Basics](/tutorials/tailwind/tailwind-basics)

3. **Interactive Learning**

   - Each tutorial includes an interactive editor
   - Try code examples in real-time
   - Experiment and learn from mistakes

4. **Track Your Progress**
   - Complete exercises at your own pace
   - Build a portfolio as you learn
   - Apply concepts to real projects
